# 🚀 Project #2: Dynamic Portfolio Modal

A reusable UI component built as part of **Jonas Schmedtmann's "The Complete JavaScript Course 2025"**.

While the core logic was learned through the course, I have refactored and upgraded this project to meet modern development standards, focusing on clean architecture and improved User Experience (UX).

## 🛠️ Tech Stack & Key Features

- **Vanilla JavaScript:** Efficient DOM manipulation and global keyboard event listeners (Esc key).
- **BEM Methodology:** Implemented Block Element Modifier naming for scalable and maintainable CSS.
- **Dynamic Data Injection:** Leveraged **HTML5 Data Attributes** (`dataset`) to update modal content dynamically from the HTML. This allows a single modal to handle multiple data sets.
- **Smooth UX/UI:** Implemented CSS transitions (`opacity` and `transform`) and `backdrop-filter` for a polished, "vaunted" feel.

## 💡 What I Learned

- **DOM Caching:** Selecting elements once and storing them in variables to improve performance.
- **NodeLists vs. Arrays:** Using `querySelectorAll` to loop through multiple buttons with a single event listener logic.
- **Refactoring:** Organizing code into helper functions to keep the codebase **DRY** (Don't Repeat Yourself).
- **Component Architecture:** Thinking of the UI as independent, reusable pieces rather than just a static page.
